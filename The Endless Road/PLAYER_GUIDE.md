# 🌲 THE ENDLESS ROAD — Quick Start Guide

## 🎮 The Game in 60 Seconds

Trapped in a town. Monsters come at night. Grab a **Talisman**, get inside, fix a **Radio**, find a **Code**, and escape. Easy? Not when you're losing your mind.

---

## ⚡ Key Stats

| Stat | Start | Danger | Death |
|------|-------|--------|-------|
| ❤️ Health | 100 | Night attacks: -15-40 | ≤ 0 |
| 🧠 Sanity | 100 | Events: -5-25 | ≤ 0 |
| ⏳ Moves | 5 per day | Night = monsters! | Time up |

---

## 🕐 Time System

```
Move 1 ──► Move 5 [NIGHT FALLS] ──► Move 10 [SUN RISES]
 ☀️ DAY    🌙 NIGHT (DANGEROUS!)   ☀️ NEW DAY
```

**FREE actions** (don't use moves): `look`, `inventory`, `status`  
**PAID actions** (use 1 move): `go`, `take`, `search`, `talk`, `invite`, `hang talisman`, `fix radio`, `broadcast`

---

## 🗺️ 9 Locations (Pick 5!)

```
🏙️ TOWN CENTER ──┬─→ 🏚️ HOUSE (Safehouse)
                 ├─→ ⛪ CHURCH (Talisman + Diary 1)
                 ├─→ 🍽️ DINER (Cook NPC + Diary 2)
                 ├─→ ⛽ GAS STATION (Copper Wire + Diary 3)
                 ├─→ 🌲 FOREST (Antenna Piece)
                 ├─→ 🕳️ CAVE (⚠️ 3-TURN LIMIT! Parts here)
                 ├─→ 🚔 SHERIFF (Sheriff NPC + Diary 4)
                 └─→ 📻 HILL (Finish line!)
```

## ⚠️ The 3 Death Traps

1. **CAVE** — Only 3 turns inside or monsters wake (instant death!)
2. **TRAITOR** — One NPC is fake. Invite them at night = open window = monsters = death
3. **NIGHT OUTSIDE** — Being outside after 5 moves = attacked = major damage

---

## 🧠 Sanity < 50 = HALLUCINATION MODE
When sanity drops below 50:
- Descriptions lie to you
- Traitor looks friendly
- Your mind betrays you

**Fix**: Invite a HUMAN NPC for +10 sanity

---

## 💀 How to NOT Die

| Situation | Action | Why |
|-----------|--------|-----|
| It's night | Be inside + have talisman hung | Only way to stay safe |
| Low health | Use bandages/first aid kit | Health = life |
| Low sanity | Invite friendly NPC | Sanity < 50 = game tricks you |
| In cave 3+ turns | GET OUT NOW | Monsters wake = instant death |
| Unsure who's traitor | Talk to both, then invite the honest one | Wrong choice = death |

## 🎯 Essential Commands

```bash
go [place]        # Move to a location
take [item]       # Grab something
search            # Find hidden stuff
hang talisman     # Secure inside location
talk              # Ask NPC questions
invite            # Bring NPC to safehouse (⚠️ choose wisely!)
fix radio         # Needs 4 parts, must be at HILL
broadcast 4275    # Send code, must have fixed radio
```

**Type `help` in-game for all commands**
